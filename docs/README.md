## üì¶ Modules Overview

### Poets Module
The poetry management system allows you to parse, search, and manage collections of poems. Perfect for building poetry databases and analyzing literary works.

**Key Features:**
- Parse structured poem data
- Search by poet name
- Filter by year or year range
- Add new poems dynamically

### Codes for Fun Module
A comprehensive collection of utility functions covering mathematical operations, string manipulation, and practical tools.

**Key Features:**
- Mathematical functions (factorial, fibonacci, prime checking, etc.)
- String manipulation (reverse, palindrome checking, case conversion, etc.)
- BMI calculator
- Account and password generators

### Random Range Module
Generate and visualize random numbers with statistical analysis capabilities.

**Key Features:**
- Flexible random number generation
- Statistical calculations
- Multiple visualization types (line, bar, scatter)
- Support for unique samples
