# Examples Directory

This directory contains demonstration scripts showcasing the functionality of each module in the Python Code Collection.

## Available Examples

### 1. Poetry Management Demo (`poets_demo.py`)

Demonstrates the poetry management system capabilities:
- Parsing poem data from formatted strings
- Displaying all poems in a collection
- Searching for poems by specific poet
- Filtering poems by year or year range
- Adding new poems to the collection

**Run:**
```bash
python examples/poets_demo.py
```

### 2. Utility Functions Demo (`utility_functions_demo.py`)

Comprehensive demonstration of utility functions:
- Mathematical operations (factorial, fibonacci, prime checking, GCD, power)
- String manipulation (reverse, case conversion, palindrome checking, vowel counting)
- BMI calculations with health categories
- Account and password generation
- Character counting in text

**Run:**
```bash
python examples/utility_functions_demo.py
```

### 3. Visualization Demo (`visualization_demo.py`)

Shows random number generation and visualization:
- Basic random number generation with different ranges
- Statistical analysis of generated data
- Different visualization types (line, bar, scatter plots)
- Comparison of unique vs non-unique samples
- Advanced usage scenarios

**Run:**
```bash
python examples/visualization_demo.py
```

**Note:** Requires matplotlib for visualizations. Install with:
```bash
pip install matplotlib
```

## Running All Examples

To run all examples in sequence:

```bash
python examples/poets_demo.py
python examples/utility_functions_demo.py
python examples/visualization_demo.py
```
